Sneaker Finder

Banco de dados:
Fluxo básico do sistema:

1. Receber arquivos (CSV ou XML)
2. Processar os dados (converter para JSON e validar)
3. Inserir os dados no banco de dados (MySQL ou PostgreSQL)
4. Criar uma API no backend (Node.js + Express)
   * Rotas: GET, POST, PUT, DELETE
5. Consumir a API no frontend (fetch)
6. Renderizar os dados dinamicamente no HTML

Observações importantes:

* Não converter diretamente para SQL manualmente
* Não usar Excel como banco de dados
* HTML deve ser dinâmico (gerado via JavaScript)

Fluxo final recomendado:
CSV/XML → Backend → Banco de Dados → API → Frontend → HTML
